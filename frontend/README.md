# ChambuaViSmart Frontend

Angular app scaffold with routing and pages.

- Run dev server: npm install && npm start
- Build: npm run build

Proxy: /api -> http://localhost:8082

Routes:
- Home (/)
- Fixtures (/fixtures)
- Teams (/teams)
- Matchup Analyzer (/matchup)
- League Table (/league)
- xG History (/xg)
- Advice (/advice)
- Match History (/history)
- Admin Upload (/admin)
